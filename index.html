About me

My name is Martin and I am an aspiring web developer and software engineer, chasing my passion for design, coding and software development. I have a wide variety of experience in many different subjects and a proven track record of academic achievement in a wide variety of fields, including cybersecurity.

Prior to Joining the consultancy program at _Nology I completed an internship in machine learning at Ant61, and I graduated from coder academies web development bootcamp.

Miguelitto's Pizzeria

Miguelitto’s Pizzeria is a local food store that offers a varied menu of classic and popular pizza types and flavours, as well as its “Build your own pizza” service, where customers can create custom pizza flavour using any of the available bases, sauces and toppings.

Seagull!

This was the first game I built and published with the Unity Engine (and it shows), with the goal of gaining some experience with as much of the game development pipeline as possible. I did all of the programming, 3D modelling, rigging and animating myself; in fact the only thing I didn't do was the sound composition! Unfortunately, this was before I discovered Git and github, so there is no history or source code available, but I have uploaded the completed .apk file to github for you to download. It should run on any android device.
